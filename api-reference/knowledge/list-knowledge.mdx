# List Knowledge

Retrieve all knowledge base items belonging to the authenticated project.

## Endpoint

```
GET /v1/knowledge
```

## Request

```bash
curl -X GET "https://api.callem.ai/v1/knowledge" \
  -H "Authorization: Bearer ck_live_your_api_key_here"
```
